## 📖 About

Bishop Chessboard Visualizer is an interactive web application that visualizes all possible moves of a bishop on a customizable chessboard. Users can generate boards of different sizes, select any square, and instantly view every valid bishop move based on chess movement rules.

The project demonstrates algorithm implementation, dynamic DOM rendering, event handling, responsive UI design, and clean component-based architecture while providing an intuitive and visually polished user experience.
